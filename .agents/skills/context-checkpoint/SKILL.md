# Context Checkpoint

Create a compact handoff with:

1. Objective
2. Current source-of-truth files
3. Decisions made
4. Files touched
5. Validation run and results
6. Known blockers
7. Explicit non-goals
8. Next smallest step

Do not paste full logs, full patient cards, or full prompts unless the exact
content is the subject of the handoff.
