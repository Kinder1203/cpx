# Repository Skills

Repo-local Skills are deliberately small and task-routed. Durable policy remains
in `README.md` / `AGENTS.md`; current state remains in `.codex/state/`.

## Active Skills

- `cpx-orient`: compact active-track orientation.
- `pre-implementation-grill`: align intent before substantial design or app work.
- `cpx-guard`: CPX safety and contract drift guard.
- `prompt-harness`: patient-card and prompt harness workflow.
- `diagnose-failure`: evidence-based failure diagnosis.
- `state-hygiene`: maintain state placement and compactness.
- `context-checkpoint`: preserve long-thread working context.
- `app-product-spec`: PRD, functional spec, user-flow, and wireframe planning.
- `app-design-system`: UI design contract and Figma design-system intake.
- `app-ui-qa`: app UI visual and interaction QA routing.

Do not load every Skill for an ordinary task. Use the smallest one matching the
current objective.
